# Skill Template

Use this template when creating new skills in `.claude/skills/`.

## Directory Structure

```
.claude/skills/
└── skill-name/
    └── SKILL.md
```

## Template Structure

```markdown
# Skill Name

[Brief description of the skill - what expertise it provides]

## Activation

- Condition 1
- Condition 2
- Condition 3

[Explanation of core concept]

// Code example demonstrating concept
```

[Explanation of core concept]

```javascript
// Code example demonstrating concept
```

```javascript
// Complete pattern implementation
```

```javascript
// Complete pattern implementation

## Best Practices

1. **Focused Expertise**: Each skill covers ONE domain
2. **Practical Examples**: Include runnable code examples
3. **Error Guidance**: Document common errors and fixes
4. **Performance Aware**: Include relevant performance targets
5. **Integration Points**: Document how skill connects to codebase
