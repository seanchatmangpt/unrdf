# Welcome to UNRDF

**Universal RDF Framework** - A next-generation toolkit for building knowledge graphs with RDF, event sourcing, and hyperdimensional reasoning.

## What is UNRDF?

UNRDF is a comprehensive ecosystem for working with RDF (Resource Description Framework) data in modern JavaScript/TypeScript applications. It combines:

- **Modern RDF Stack**: Built on Oxigraph (Rust-based WASM triple store)
- **React Integration**: 40+ React hooks for seamless UI integration
- **Event Sourcing**: Complete audit trail with time-travel capabilities
- **Hyperdimensional Reasoning**: KGC-4D for multi-dimensional knowledge composition
- **Browser & Server**: Isomorphic design works everywhere

## Quick Start

```bash
# Install core package
pnpm add @unrdf/core

# Or use React hooks
pnpm add @unrdf/hooks

# Or use KGC-4D
pnpm add @unrdf/kgc-4d
```
