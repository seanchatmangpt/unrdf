# @unrdf/oxigraph

Low-level Oxigraph WASM bindings. Most users should use `@unrdf/core` instead.

## Installation

```bash
pnpm add @unrdf/oxigraph
```

## Quick Start

```javascript
import { createStore, dataFactory } from '@unrdf/oxigraph';

const store = createStore();
const quad = dataFactory.quad(/* ... */);

store.add(quad);
```

---

## When to Use

Use `@unrdf/oxigraph` directly when you need:
- Direct Oxigraph API access
- Custom WASM configuration
- Performance-critical paths

For most use cases, use `@unrdf/core` instead.
