# @unrdf/browser

Browser-optimized RDF bundle with IndexedDB persistence and streaming support.

## Installation

```bash
pnpm add @unrdf/browser
```

## Features

- **IndexedDB Storage**: Persistent RDF data in the browser
- **Streaming Parser**: Incremental N3/Turtle parsing
- **Optimized Bundle**: Tree-shaken for minimal size

## Quick Start

```javascript
import { BrowserStore } from '@unrdf/browser';

const store = new BrowserStore('my-knowledge-graph');
await store.ready;

// Works like createStore() but persists to IndexedDB
store.add(quad);
```
