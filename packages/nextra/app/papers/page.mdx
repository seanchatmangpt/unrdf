### Hyperdimensional Information Theory for RDF

<Tabs items={['Abstract', 'Key Results', 'Citation']}>
<Tabs.Tab>
**Hyperdimensional Computing Meets Semantic Web: Information-Theoretic Foundations**

We explore the application of hyperdimensional computing (HDC) to RDF knowledge graphs, demonstrating how high-dimensional vector representations enable efficient similarity search, analogical reasoning, and concept composition. We prove information-theoretic bounds on storage capacity and provide algorithms for encoding RDF graphs in hyperdimensional space.

[Read Paper →](/papers/hyperdimensional-reasoning)
</Tabs.Tab>
<Tabs.Tab>
**Theorem 1 (Storage Capacity)**:

A hyperdimensional vector space of dimension $d$ can reliably store:

```math
N \leq \frac{d}{2 \log_2 d}
```

independent concepts with cosine similarity below $\epsilon$.

**Theorem 2 (Binding Distributivity)**:

For vectors $\vec{a}, \vec{b}, \vec{c} \in \mathbb{R}^d$:

```math
(\vec{a} + \vec{b}) \odot \vec{c} \approx \vec{a} \odot \vec{c} + \vec{b} \odot \vec{c}
```

where $\odot$ is element-wise multiplication (binding).

### Event Sourcing for RDF Graphs

<Tabs items={['Abstract', 'Performance', 'Citation']}>
<Tabs.Tab>
**Immutable Event Logs as the Foundation for Temporal RDF Stores**

This paper presents a formal model for event-sourced RDF stores, where all state changes are recorded as immutable events rather than in-place updates. We prove correctness of temporal queries, analyze space-time tradeoffs, and demonstrate a production implementation achieving 40% faster queries with 60% less memory than traditional triple stores.

[Read Paper →](/papers/event-sourcing-rdf)
</Tabs.Tab>
<Tabs.Tab>
**Benchmark Results** (vs. traditional RDF stores):

| Metric | KGC-4D Event Sourcing | Traditional Store | Improvement |
|--------|----------------------|-------------------|-------------|
| Query Speed | 50ms (avg) | 83ms (avg) | **40% faster** |
| Memory Usage | 42MB | 105MB | **60% less** |
| Time-Travel | $O(m \log n)$ | $O(n)$ full scan | **Logarithmic** |
| Audit Trail | Built-in | Requires plugin | **Native** |

Where $m$ = events to replay, $n$ = total quads.

## Advantages of Web-Readable Papers

### Traditional PDF
- ❌ Math formulas as images (not searchable)
- ❌ Fixed layout (poor mobile experience)
- ❌ No inline code execution
- ❌ Citations require manual lookup
- ❌ Difficult to copy formulas

### Web-Readable (This Site)
- ✅ **Searchable Math**: LaTeX rendered as text with KaTeX
- ✅ **Responsive**: Works on all devices
- ✅ **Interactive**: Runnable code examples
- ✅ **Linked Citations**: Click to navigate
- ✅ **Copy-Paste Formulas**: Select and copy LaTeX

<Callout type="info">
**Example**: Try selecting this formula: $\int_2^3 x^3 \, dx = \left[\frac{x^4}{4}\right]_2^3 = \frac{65}{4}$

You can copy it as LaTeX!
</Callout>
