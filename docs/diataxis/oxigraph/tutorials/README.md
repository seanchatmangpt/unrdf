# Tutorials — @unrdf/oxigraph

Tutorials are learning-oriented. Work through them in order and you will build real working examples.

## Available Tutorials

| #   | Title                                                         | Time   | What you build                                         |
| --- | ------------------------------------------------------------- | ------ | ------------------------------------------------------ |
| 01  | [Build your first Oxigraph store](01-persistent-rdf-store.md) | 15 min | A populated store, SPARQL queries, and a Turtle export |

## Before You Start

- Node.js 18 or later installed
- pnpm available (`npm install -g pnpm`)
- Basic familiarity with JavaScript ESM (`.mjs`)

You do not need prior knowledge of RDF or SPARQL — each tutorial explains the concepts as it introduces them.

## After the Tutorials

Once you have completed the tutorial, move to the [How-To Guides](../how-to/README.md) for task-specific recipes, or to the [Reference](../reference/README.md) for the complete API surface.
