# How-to Guides

How-to guides are goal-oriented. They assume you already know the basics (see [tutorials](../tutorials/README.md)) and show you how to accomplish a specific task.

## Guides in This Section

| Guide                                                           | Goal                                                                                               |
| --------------------------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| [01 — Execute Hooks from the CLI](01-execute-hooks-from-cli.md) | Run the full hook workflow: generate a template, validate it, and execute it against an RDF store  |

## Scope

These guides focus on `unrdf hooks`. For graph and code-generation workflows see [reference/commands.md](../reference/commands.md).
