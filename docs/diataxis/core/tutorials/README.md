# Tutorials — @unrdf/core

Tutorials are step-by-step guides for learners. They walk you through building something
concrete and teach through doing. You do not need prior knowledge of `@unrdf/core` to follow
these tutorials.

## Index

| Tutorial                                                        | What you build                                   |
| --------------------------------------------------------------- | ------------------------------------------------ |
| [01 — Store and query RDF data](./01-store-and-query-rdf.md)    | A knowledge graph of people, queried with SPARQL |
| [02 — Validate a graph with SHACL](./02-validate-with-shacl.md) | A shape definition and a validation report       |

## Prerequisites

```bash
pnpm add @unrdf/core
```

Node.js 18 or later is required.
