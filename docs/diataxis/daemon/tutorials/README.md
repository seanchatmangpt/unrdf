# Tutorials: @unrdf/daemon

Tutorials are **learning-oriented**. They guide you through a concrete task to build understanding of the daemon, without requiring you to know the full API up front.

## Available Tutorials

| Tutorial                                  | What you learn                                                     | Time   |
| ----------------------------------------- | ------------------------------------------------------------------ | ------ |
| [Your First Daemon](./01-first-daemon.md) | Create a daemon, schedule an operation, observe results via events | 15 min |

## Before You Start

All tutorials assume:

- Node.js 18+ with ESM support (`"type": "module"` in package.json or `.mjs` extension)
- `@unrdf/daemon` installed: `pnpm add @unrdf/daemon`
- Basic async/await familiarity

## After the Tutorials

Once you have completed the tutorials:

- Use the [How-To Guides](../how-to/) for targeted task solutions
- Consult the [API Reference](../reference/daemon-api.md) for full method signatures
- Read the [Architecture Explanation](../explanation/01-daemon-architecture.md) for design reasoning
