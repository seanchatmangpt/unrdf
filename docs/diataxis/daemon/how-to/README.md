# How-To Guides: @unrdf/daemon

How-to guides are **task-oriented**. Each guide solves one specific problem. Unlike tutorials, they assume you already know the basics and need a direct answer.

## Available Guides

| Guide                                              | Solves                                                                     |
| -------------------------------------------------- | -------------------------------------------------------------------------- |
| [Schedule Operations](./01-schedule-operations.md) | Register, execute, list, and remove scheduled operations                   |
| [Enable OTel Tracing](./02-enable-otel-tracing.md) | Set `OTEL_ENABLED=true`, configure the OTLP endpoint, read MCP tool spans  |
| [Configure Security](./03-configure-security.md)   | API key authentication, injection detection, environment-aware enforcement |

## Before Using These Guides

Complete the [tutorial](../tutorials/01-first-daemon.md) first if you have not used `@unrdf/daemon` before.

For full API signatures, see the [Reference](../reference/).
