# Tutorials

Tutorials are learning-oriented. They guide you through a concrete task from start to finish so you can build a working mental model of the CLI. You do not need prior knowledge of UNRDF to follow them.

## Prerequisites

- `unrdf` installed: `pnpm add -g @unrdf/cli`
- Node.js >= 18

## Tutorials in This Section

| Tutorial                                            | What You Learn                                                                                                      |
| --------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| [01 — Query an RDF Graph](01-query-an-rdf-graph.md) | Load a Turtle file into an in-memory graph, run a SPARQL SELECT query, run a CONSTRUCT query, and export to JSON-LD |

## Next Steps After Tutorials

Once you have finished a tutorial, the how-to guides show you how to accomplish specific goals (such as running hooks or validating conditions). The reference section documents every flag precisely.
