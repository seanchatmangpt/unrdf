### `@unrdf/otel`

**Description:** OpenTelemetry semantic convention constants for UNRDF, generated by OTel Weaver from `otel/registry/`. Import these instead of raw strings to catch typos at lint time.

**Exports:**

```javascript
// All constants (re-exports both modules)
import { ATTR_MCP_TOOL_NAME, METRIC_MCP_TOOL_DURATION } from '@unrdf/otel';

// Attribute name constants only
import {
  ATTR_MCP_SERVER_NAME, // 'mcp.server.name'
  ATTR_MCP_TOOL_NAME, // 'mcp.tool.name'
  ATTR_MCP_TOOL_SUCCESS, // 'mcp.tool.success'
  ATTR_MCP_TOOL_RESULT_SIZE, // 'mcp.tool.result_size'
  ATTR_MCP_TOOL_ERROR_TYPE, // 'mcp.tool.error_type'
  ATTR_UNRDF_CLI_COMMAND, // 'unrdf.cli.command'
  ATTR_UNRDF_CLI_SUBCOMMAND, // 'unrdf.cli.subcommand'
  ATTR_UNRDF_CLI_EXIT_CODE, // 'unrdf.cli.exit_code'
  ATTR_UNRDF_HOOK_CONDITION_KIND, // 'unrdf.hook.condition_kind'
  ATTR_UNRDF_HOOK_SATISFIED, // 'unrdf.hook.satisfied'
  ATTR_UNRDF_RDF_OPERATION, // 'unrdf.rdf.operation'
  ATTR_UNRDF_RDF_STORE_BACKEND, // 'unrdf.rdf.store_backend'
  // ... 44 ATTR_* constants total
} from '@unrdf/otel/attributes';

// Metric name constants only
import {
  METRIC_MCP_TOOL_DURATION, // 'mcp.tool.duration'
  METRIC_MCP_TOOL_INVOCATIONS, // 'mcp.tool.invocations'
  METRIC_UNRDF_HOOK_EVALUATIONS, // 'unrdf.hook.evaluations'
  METRIC_UNRDF_RDF_QUAD_COUNT, // 'unrdf.rdf.quad_count'
  METRIC_UNRDF_TRANSACTION_DURATION, // 'unrdf.transaction.duration'
  // ... 13 METRIC_* constants total
} from '@unrdf/otel/metrics';
```

**Source:** `packages/otel/src/generated/` — do not edit directly.

**Re-generate:**

```bash
cd otel && weaver registry generate --registry registry/ js ../packages/otel/src/generated/
```

**Registry:** `otel/registry/` — 43 attributes, 13 metrics across 8 YAML files.
